# Media elements

## iframe

Remove *iframe* border.

* **Type**: reset
* **Target**: all

```css
iframe {
	border: 0;
}
```

## iframe, img, input, video, select, textarea

Reset size to all media elements.

* **Type**: reset
* **Target**: all

```css
iframe, img, input, video, select, textarea {
	height: auto;
	width: 100%;
}
```
