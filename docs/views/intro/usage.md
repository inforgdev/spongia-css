# Usage

> **Note:** the import need to be the first CSS import.

## CSS Loader

Using a CSS loader, in JS:

```js
import 'spongia-css'
```

## Vanilla CSS

```scss
@import 'node_modules/spongia-css/dist/spongia.min.css'
```

## Vanilla HTML

```html
<link rel="stylesheet" href="node_modules/spongia-css/dist/spongia.min.css">
```
